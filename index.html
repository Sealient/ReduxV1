Introduction

ReduxV1 is a powerful, lightweight UI library for Roblox that provides a clean, modular interface system with extensive customization options.

Customizable Theme

Built-in theme system with color pickers, transparency controls, and text size options.

Lightweight

Optimized performance with minimal impact on your game's performance.

Modular Design

Easy to extend and customize with a clean, organized code structure.

Installation

Load ReduxV1 directly into your Roblox game using a single line of code:

LocalScript
-- Paste this in a LocalScript
local ReduxV1 = loadstring(game:HttpGet("https://raw.githubusercontent.com/Sealient/ReduxV1/refs/heads/main/ReduxV1.lua"))()

Quick Start

Create your first UI in just a few lines of code:

Basic Example
-- Load the library
local ReduxV1 = loadstring(game:HttpGet("https://raw.githubusercontent.com/Sealient/ReduxV1/refs/heads/main/ReduxV1.lua"))()

-- Create main UI window
local UI = ReduxV1:CreateMain("MyUI")

-- Create a tab
local MainTab = UI:CreateTab("Main")

-- Add a button
MainTab:CreateButton("Click Me", "This is a test button", function()
    print("Button clicked!")
end)

Complete Example

Here's a complete example showcasing all UI elements:

Complete Example
-- Complete ReduxV1 Example
local ReduxV1 = loadstring(game:HttpGet("URL_TO_SCRIPT"))()
local UI = ReduxV1:CreateMain("ExampleUI")

-- Main Tab
local MainTab = UI:CreateTab("Main")
MainTab:CreateLabel("Welcome!", Color3.fromRGB(0, 255, 255))
MainTab:CreateButton("Test Button", "Click me", function()
    print("Working!")
end)

-- Player Tab
local PlayerTab = UI:CreateTab("Player")
PlayerTab:CreateSlider("WalkSpeed", 16, 200, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

-- Settings tab is automatically added!

API Reference

ReduxV1:CreateMain(title)

Creates the main UI window. Returns a UI object.

Parameter Type Description
title string Window title

UI:CreateTab(name)

Creates a new tab. Returns a Tab object.

Parameter Type Description
name string Tab display name

Tab Methods

Method Parameters Description
CreateButton text, tooltip, callback Creates a clickable button
CreateToggle text, tooltip, callback Creates a toggle switch
CreateSlider text, min, max, default, callback Creates a value slider
CreateColorPicker text, defaultColor, callback Creates a color picker
CreateSideDropButton text, options, callback Creates a dropdown menu
CreateLabel text, color Creates a text label
CreateTextBox text, placeholder, callback Creates a text input field

Built-in Settings

ReduxV1 automatically includes a Settings tab with:

  • Background transparency control
  • Text color picker (Green, Blue, Red, Yellow, White)
  • Text size toggle (14px / 16px)
  • Reset to defaults button
  • Destroy UI button
